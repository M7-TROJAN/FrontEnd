Attribute Selectors

Attribute selectors are used to select elements based on the presence of an attribute or the value of an attribute.

For example, the following selector selects all input elements with a type attribute value of "text":

        input[type="text"] {
            background-color: yellow;
        }

Here are some examples of attribute selectors:

  • [attribute=value]
  • [attribute~=value]
  • [attribute|=value]
  • [attribute^=value]
  • [attribute$=value]
  • [attribute*=value]

Example

Here is an example of using attribute selectors:

        <style>
            input[type="text"] {
                background-color: yellow;
            }
            input[type="password"] {
                background-color: lightblue;
            }
            input[type="submit"] {
                background-color: lightgreen;
            }
        </style>

Try changing the type attribute value of the input elements to see the effect of the attribute selectors.
